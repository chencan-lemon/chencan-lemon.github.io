Windows系统下搭建个人博客

Posted on 2020-07-07

title:Windows系统下搭建个人博客
date:2020-07-07 11:00:33
tags:Win10 Hexo GitHub

一、安装Hexo框架https://hexo.io/

​ 1.安装依赖环境

​ 1.1 下载Node.js 官网地址https://nodejs.org/zh-cn/ 选择LTS长期支持版

​ 1.2 下载git 官网地址https://git-scm.com/

​ 2.安装Hexo

​ 在刚才下载好的git文件下，找到git-bash.exe并运行，会弹出一个窗口，以下命令都在此窗口中操作

​ 2.1 借助淘宝的镜像

​ $ npm install -g cnpm –registry=https://registry.npm.taobao.org

​ 2.2 安装hexo

​ $ cnpm install -g hexo

​ 2.3 创建文件夹

​ E:\blog\chencan，存放博客内容的地方

​ 2.4 初始化文件夹

​ 2.4.1 进入文件夹

​ $ cd /e/blog/chencan

​ 2.4.2 初始化

​ $ hexo init

​ 2.5 再一次安装hexo,会继续下载一些文件

​ $ cnpm install

​ 2.6 生成站点

​ $ hexo g

​ 2.6 启动站点

​ $ hexo g

​ 3. 在浏览器地址栏输入http:localhost:4000

二、配置GitHub

​ 1.注册GitHub账号，有则可以忽略

​ 2.创建GitHub仓库(repository)

​ 仓库名称最好是GitHub用户名.github.io，为了以后可以直接使用http://GitHub用户名.github.io访问博客

​ 3.SSH公钥和私钥

​ 运行git-bash.exe,使用$ cd~/.ssh查看本机是否存在ssh的密钥，如果第一次使用，那么会显示No such file or directory，

​ 3.1 这时输入

​ $ ssh-keygen -t rsa -C “GitHub邮件地址”

​ 会在C:\Users\lemon生成一个.ssh文件夹

​ 3.2 打开.ssh文件夹，将id_rsa.pub文件里的内容(密钥)拷贝到 GitHub个人主页–>Settings–>SSH and GPG keys –>New SSH key–>Key，title随便填写，点击Add SSH key按钮，添加即可

​ 3.3 测试是否添加成功

​ $ ssh -T git@github.com # 注意邮箱地址不用改

​ 提示Are you sure you want to continue connecting (yes/no)?，输入yes,会提示Hi Sanarous! You’ve successfully authenticated, but GitHub does not provide shell access.

​ 说明SSH添加成功

​ 3.4 如果本机不是第一次使用git接github，则只需要测试SSH是否添加成功

  1. Git提交的用户信息

    $ git config – global user.name “GitHub用户名”

    $ git config – global user.email “GitHub邮箱地址”

三、通过Hexo框架将博客部署到GitHub上

​ 1.打开E:\blog\chencan_config.yml的配置文件，找到deploy并补全

​ deploy:

type: git

​ repo: git@github.com:GitHub用户名/GitHub用户名.github.io.git

​ branch: master

2.运行git-bash.exe

​ $ cd /e/blog/chencan

​ 3.安装一个部署需要的插件

​ $ npm install hexo-deployer-git –save

​ 4.部署到GitHub上

​ 4.1 清除缓存

​ $ hexo clean

​ 4.2 本地生成博客

​ $ hexo g

​ 4.3 部署

​ $ hexo deploy

​ 5.部署一篇新的博客

​ 5.1 $ hexo new post “文件名”

​ 5.2 用对markdown件友好的编辑器对文章进行编辑，比如 MarkdowmPad、Typora等

​ 5.3 $ hexo clean && hexo g && deploy

​ 6.部署完成后，在浏览器地址栏输入

​ https://github.com/chencan-lemon/chencan-lemon.github.io

​ 可以看到发布了的博客

四、Hexo框架的Next主题

​ 1.在GitHub上搜索hexo-next

​ 2.下载Next主题

​ 运行git-bash.exe

​ 2.1 $ cd /e/blog/chencan

​ 2.2 $ git clone https://github.com/next-theme/hexo-theme-next themes/next

​ 3.配置博客主题为Next主题

​ 打开E:\blog\chencan_config.yml的配置文件，将theme设置为next

​ 4.测试Next主题

​ 4.1 清除缓存

​ $ hexo clean

​ 4.2 本地生成博客

​ $ hexo g

​ 4.3 启动站点

​ $ hexo s

​ 5.在浏览器地址栏输入http:localhost:4000

五、Hexo框架的设置

​ 1.全局配置文件

​ E:\blog\chencan_config.xml

​ 2.主题配置文件

​ E:\blog\chencan\themes_config.xml
